# Lazygit Configuration

**Note:** The actual lazygit configuration is stored in `~/.dotfiles/.config/lazygit/` (not managed by stow).

Since `~/.config` is symlinked to `~/.dotfiles/.config`, the lazygit config is located at:
- Config: `~/.dotfiles/.config/lazygit/config.yml`
- Accessible via: `~/.config/lazygit/config.yml`

See `~/.config/lazygit/README.md` for full documentation.

## Quick Access

To edit the config:
```bash
$EDITOR ~/.config/lazygit/config.yml
```

To view the config documentation:
```bash
cat ~/.config/lazygit/README.md
```
